# Work_Os

#### A workOs made in Flutter with using firebase..

## Features

* Add Tasks
* Show All Workers
* Show Tasks Details
* Show User State
* Login
* Register
* Show User Profile
* Forget Password

## Installation

### 1. Clone the repo

```
git clone https://github.com/MahmoudDabour1/workOS.git
cd workOs Application 
```

### 2. Run pub get

```
flutter pub get
```
### 3. Run the app

```
flutter run
```
